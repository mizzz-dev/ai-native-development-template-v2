# Ai Context Layering

このドキュメントは commercial-ai-native-engineering-os の運用基準です。

## 目的
- 実務で再利用できる判断基準を提供する。
- AIと人間の責務分離を維持する。
- 商用運用・監査・復旧性を担保する。

## 必須ルール
- 変更理由、影響、ロールバック、検証結果を日本語で記録する。
- 関連Issue/PR/ADR/Discussionを相互リンクする。
- 会話のみで意思決定を完了させず、Repositoryに残す。

## 運用チェック
- [ ] 記録が日本語で残っている
- [ ] 監査に必要な証跡がある
- [ ] 顧客影響評価がある
