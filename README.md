# ai-native-development-template-v2

OSSとしてAI Native開発を支援するテンプレート/CLI集です。

## Quick Start
1. `npm install`
2. `npm run build:create-ai-native-dev`
3. `node packages/create-ai-native-dev/dist/cli.js init --dry-run`
