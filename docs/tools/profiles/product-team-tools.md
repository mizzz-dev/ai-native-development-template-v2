# product-team-tools

## 推奨ツール
- 主要カテゴリから最小構成を選択

## 任意ツール
- チーム事情に応じて追加

## 避けるべきツール
- 監査不能、承認不能、秘密管理不備

## 代替ツール
- self-host / OSS / internal platform

## 導入条件
- 人間承認、証跡保存、撤退計画

## 証跡要件
- `docs/templates/tool-integration-evidence-template.md`

## MCP利用可否
- read-only推奨、write承認必須

## 外部SaaS可否
- プロファイルに応じて制限

## data classification注意点
- 機密データ投入範囲を明示
