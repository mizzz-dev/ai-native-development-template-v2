# Use Cases

READMEの概要を補完する、想定ユースケース集です。

## 1. 個人開発で使う
- 目的: AIと小さく素早く作る
- 推奨レベル: Level 1
- 主要ファイル: `PROMPT.md`, `docs/templates/`, `docs/logs/`

## 2. 学生チームで使う
- 目的: 課題制作で役割分担とレビューを回す
- 推奨レベル: Level 2
- 主要ファイル: `.github/`, `docs/core/`, `docs/adr/`

## 3. 非エンジニアがAIに依頼する
- 目的: 仕様をIssue化してAIに実装依頼
- 推奨レベル: Level 1
- 主要ファイル: `PROMPT.txt`, `docs/adoption-guide.md`, `docs/templates/`

## 4. 小規模チームで使う
- 目的: PR/Issue/commit運用の統一
- 推奨レベル: Level 2
- 主要ファイル: `.github/pull_request_template.md`, `.github/ISSUE_TEMPLATE/`, `docs/core/`

## 5. 商用サービスで使う
- 目的: 品質・セキュリティ・運用を含めた継続提供
- 推奨レベル: Level 3
- 主要ファイル: `docs/core/commercial-readiness.md`, `docs/core/security-rules.md`, `docs/runbooks/incident-response.md`
