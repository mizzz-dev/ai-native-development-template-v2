# screenshots

GitHub Issue/PR運用例や導入手順の画面キャプチャを置くディレクトリです。

## 何を置く場所か
- Issue作成例
- PRテンプレート入力例
- 導入チェック結果の画面

## READMEからどう参照するか
- 実ファイル追加後に、READMEまたはdocsから相対リンクで参照します。
- 画像が未追加の状態ではリンクを貼らないでください。

## 命名ルール
- 形式: `ss-<topic>-yyyy-mm-dd-vN.png`
- 例: `ss-first-issue-2026-05-13-v1.png`

## 注意
- 存在しない画像をREADMEから参照しないこと
- 個人情報、secret、アクセストークンを含めないこと
