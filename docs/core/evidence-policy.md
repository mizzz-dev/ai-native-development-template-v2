# Evidence Policy

## 目的
第三者が「何を、なぜ、どのように、どの結果で」変更したかを追跡可能にする。

## 保存先
- GitHub: 差分、PR、レビュー、CI
- Linear: 課題状態、受け入れ条件、優先度
- Notion: 作業ログ、判断補足、MCP利用履歴

## 必須記録
対象Linear Issue / 対象PR / 日時 / 作業者 / 使用AI・MCP / 入力要約 / 変更前後 / 実行コマンド / テスト結果 / CI結果 / 判断理由 / 未対応事項 / リスク / ロールバック。

## マスキング方針
secret・個人情報・顧客情報は保存前に不可逆マスキング。

## 完了条件
第三者が再現手順・判断理由・結果を確認できること。
